# AI App - Static Website

## Overview
A static HTML website with multiple learning/quiz applications in Vietnamese.

## Project Structure
- `index.html` - Main landing page with links to all applications
- `rung_chuong_vang.html` - "Ring the Golden Bell" quiz game
- `jp_it.html` - Japanese IT vocabulary quiz
- `flashcard.html` - Flashcard application
- `find_me.html` - Find me game
- `server.py` - Simple Python HTTP server to serve static files

## Running the Application
The application runs using Python's built-in HTTP server on port 5000.

```bash
python server.py
```

## Deployment
This is a static website that can be deployed as a static deployment.
