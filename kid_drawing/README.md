# Kid Drawing App

Welcome to the Kid Drawing App! This application is designed to provide a fun and interactive platform for children to draw and color. Users can create their own drawings or color existing black-and-white images, and the app includes features to track completed drawings and manage user accounts.

## Features

- **Drawing and Coloring**: Users can draw freely or color from a selection of black-and-white images.
- **Random Image Generation**: Generate random images for coloring to keep the experience fresh and exciting.
- **User Accounts**: Support for multiple user accounts, allowing children to have their own profiles with unique avatars.
- **Parent and Child Modes**: Differentiate between child and parent users. Parents can manage their children's accounts and provide oversight.
- **Avatar Management**: Parents can edit children's avatars to personalize their experience.
- **Drawing Ratings**: Parents can rate children's drawings on a scale of 1 to 5 stars.
- **Dashboard**: A dashboard to track completed drawings and provide insights into user activity.
- **Cross-Platform Compatibility**: The app is designed to work seamlessly on both desktop and mobile devices.
- **Vibrant Theme**: A lively yet balanced color scheme to engage users without overwhelming them.
- **Sound Effects**: Enjoy lively sounds and effects to enhance the drawing and coloring experience.

## Getting Started

To get started with the Kid Drawing App, follow these steps:

1. **Clone the Repository**:
   ```
   git clone <repository-url>
   cd kid_drawing
   ```

2. **Install Dependencies**:
   For the main application:
   ```
   npm install
   ```

   For the server:
   ```
   cd server
   npm install
   ```

3. **Run the Application**:
   Start the main application:
   ```
   npm run dev
   ```

   Start the server:
   ```
   cd server
   node index.js
   ```

4. **Access the App**:
   Open your browser and navigate to `http://localhost:3000` to start using the app.

## Project Structure

The project is organized as follows:

- **public/**: Contains static assets, including sound effects.
- **src/**: Contains the main application code, including components, hooks, and utilities.
- **server/**: Contains the backend server code, including routes and data management.
- **package.json**: Lists dependencies and scripts for both the client and server.
